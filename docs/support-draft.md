## Common Issues

### I cannot join a room

Check that the room code or invite link is correct. Room codes look like `BAND-482193`.

### My friend cannot see the room

Shared rooms require Supabase backend configuration. If the app is in local demo mode, rooms stay on one device.

### Spotify does not open

Install Spotify on the device, then try again. The app opens Spotify links and falls back to the Spotify web player when possible.

### Spotify search shows mock results

Real Spotify search requires a configured backend endpoint. See `docs/spotify-setup.md`.
